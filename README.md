#CELEBRIUM_RK2.0 | Portföy Vitrin

Bu proje, modern web teknolojileri kullanılarak geliştirilmiş, yüksek performanslı ve estetik odaklı bir portföy vitrinidir. **Next.js 15** ve **Tailwind CSS** kullanılarak inşa edilmiştir.

---

##  Kullanılan Teknolojiler

* **Framework:** https://nextjs.org/ (App Router)
* **Dil:** TypeScript
* **Styling:** Tailwind CSS
* **Font:** Geist (Vercel)
* **Deployment:** Vercel


Tarayıcınızda `http://localhost:****` adresine giderek projeyi görüntüleyebilirsiniz.

---

## Dosya Yapısı

* `src/`: Uygulamanın ana kaynak kodları.
* `public/`: Statik varlıklar (resimler, ikonlar).
* `cd_cd.js`: Özel fonksiyon testleri.
